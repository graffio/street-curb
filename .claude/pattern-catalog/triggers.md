# Pattern Triggers

When you see these signals, read the corresponding pattern file before proceeding.

| Signal | Read |
|--------|------|
| Collection needing iteration + lookup | `pattern-catalog/lookup-table.md` |
| if/else chain on type field | `pattern-catalog/tagged-sum.md` |
| Finite set of state changes | `pattern-catalog/action.md` |
| Complex derived state from Redux | `pattern-catalog/selector-composition.md` |
| Repeated style objects across components | `pattern-catalog/design-system-extraction.md` |
